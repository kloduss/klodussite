# GPU HUB - Інформаційний портал про графічні процесори

## Опис проекту

GPU HUB - це сучасний веб-сайт, присвячений історії та розвитку графічних процесорів. Проект створено як атестаційну роботу з використанням HTML5, CSS3 та JavaScript.

## Технології

- **HTML5** - семантична розмітка, валідний код
- **CSS3** - сучасні стилі, flexbox, grid, анімації, градієнти
- **JavaScript (ES6)** - інтерактивність, auto-hide header, кнопка "вгору"
- **Font Awesome** - іконки
- **Google Fonts** - типографіка (Space Grotesk, Inter)

## Особливості реалізації

### HTML5
- Семантичні теги (`header`, `footer`, `article`, `nav`)
- Валідна розмітка відповідно до стандартів W3C
- Meta-теги для SEO та адаптивності
- Вбудовані відео (YouTube iframe)

### CSS3
- CSS Grid та Flexbox для макетів
- CSS Variables для кольорової схеми
- @keyframes анімації (slideUpFade, float)
- Backdrop-filter для glassmorphism ефекту
- Media queries для респонсивності
- Градієнти (linear-gradient)
- Transform та transition для плавності

### JavaScript
- Vanilla JS (без фреймворків)
- Event listeners для скролу
- DOM маніпуляції
- Умовна логіка для десктоп/мобайл
- Smooth scroll behavior
